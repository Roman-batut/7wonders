# 7 Wonders Randomizer

A modern web-based randomizer for the board game "7 Wonders". Randomly assigns wonders to players and displays their seating arrangement around a circular table.

## Features

- **Player Management**: Add and remove players with an intuitive interface
- **Add-on Support**: Select from Cities, Leaders, Edifice, and Armada expansions
- **Random Assignment**: Randomly assigns wonders to players and arranges seating positions
- **Visual Display**: Beautiful animated circular table visualization showing each player's position and assigned wonder
- **Color-Coded Players**: Each player gets a unique pastel color for easy identification
- **Modern UI**: Dark theme with smooth animations and responsive design

## Usage

1. Open `seven_wonders_randomizer.html` in any modern web browser
2. Add players by entering their names and clicking "Add" (or press Enter)
3. Select any add-ons you want to include by checking the boxes
4. Click "Randomize" to randomly assign wonders and seating positions
5. View the animated table arrangement showing each player's position and wonder

## Example

1. Add players: "Alice", "Bob", "Charlie", "Diana"
2. Select "Cities" add-on
3. Click "Randomize"
4. Result: Players are randomly assigned wonders (including Byzance and Petra) and arranged around the table with animated visualization
